**Project Description**

This is a web-based Employee Management System that allows admin users to manage employee records such as adding, updating, viewing, and deleting employee details.

The application uses **Local Storage** to store and manage data on the browser.



**Features**

- Admin Login Authentication

- Add New Employees

- View Employee List

- Update Employee Details

- Delete Employees

- Data persistence using Local Storage

- Responsive UI using Bootstrap



**Technologies Used**

- HTML

- CSS (Bootstrap)

- JavaScript

- Local Storage (Browser)



**How to Run the App**

1. Extract the ZIP folder.

2. Open the project folder.

3. Double-click the `index.html` file to launch the application in any modern web browser.

4. Login using the default credentials:  

- Username: `admin`  

- Password: `admin@123`



**How Data is Stored**

- All employee data is stored in the browser using **Local Storage**.

- Data will persist even after refreshing the page.

- Clearing browser storage will remove all saved data.



**How to run the tests:**

1. Open your terminal or command prompt.

2. Navigate to the root folder of this project.

3. Run 'npm install' to install Jest.

4. Run 'npm test' to execute the test suites.
